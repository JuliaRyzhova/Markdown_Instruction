## Списки
Чтобы создать ненумерованный список нужно пункты выделить звездочкой.

Например, вот так:
* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо эти пункты просто пронумеровать. 

Например, вот так:
1. Пункт 1
2. Пункт 2
3. Пункт 3
